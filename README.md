# PWA wrapper

## Authors
- Company : Lundi8
- Code : Jonathan ATTAR

## Description
Add some features that PWA can't handle for desktop environnement
- fullscreen on launch
- automatic launch
- exit app forbidden
- wrapped for desktop

## Installation
```
npm i 
```
or
```
yarn 
```

## Scripts
```
npm start
npm run package
npm run make

```
or
```
yarn start
yarn package
yarn make
```
